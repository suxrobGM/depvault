# Secret Files

The `depvault secrets` command group manages encrypted secret files stored in the vault.

---

## secrets list

List secret file metadata for the active project.

```bash
depvault secrets list
```

| Flag | Description | Default |
| --- | --- | --- |
| `--project` | Project ID. Uses active project if omitted. | Active project |
| `--environment` | Filter by environment type. | All |
| `--output` | Output format: `table`, `json`. | `table` |

### Examples

List all secret files:

```bash
depvault secrets list
```

List production secrets as JSON:

```bash
depvault secrets list --environment PRODUCTION --output json
```
